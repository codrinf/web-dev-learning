## Internetul: cum functioneaza

Ai copiat vreodata un fisier pe un stick USB sau CD pentru a-l transfera pe alt calculator?
Daca da, nu ar fi fost de ajutor ca cele doua calculatoare sa fie conectate, pentru a transfera fisierele direct, fara nevoie de stick sau CD?

Ei bine, asta e internetul: o retea de calculatoare conectate.

Fisierele stau pe niste calculatoare, numite servere (servers), si pot fi accesate de orice alt calculator. Acestea din urma poarta numele de clienti (clients).

Sigur, resursele de mai jos vor completa aceasta definitie simpla. 

Rolul tau de programator web este sa creezi aceste fisiere, apoi sa le pui pe un server pentru a putea fi accesate de utilizatori.


### HTTP, HTTPS
Cele mai comune protocoale pe baza corara clientul comunica cu serverul. Un set de reguli de comunicare.
